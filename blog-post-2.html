Python for Data Science

# Comprehensive data cleaning example
import pandas as pd
import numpy as np

def prepare_data(df):
    # Handle missing data
    df = df.apply(lambda x: x.fillna(x.mean()) 
    
    # Feature engineering
    df['age_group'] = pd.cut(df['age'],
                           bins=[0,18,35,50,100],
                           labels=['teen','adult','mid','senior'])
    
    # Encoding categoricals
    df = pd.get_dummies(df, columns=['category'])
    
    # Normalization
    from sklearn.preprocessing import MinMaxScaler
    scaler = MinMaxScaler()
    df[['income','spending']] = scaler.fit_transform(
        df[['income','spending']])
    
    return df

SQL for Analytics

-- Advanced analytical query
WITH user_metrics AS (
  SELECT
    user_id,
    COUNT(DISTINCT order_id) AS orders,
    SUM(amount) AS lifetime_value,
    AVG(rating) AS avg_rating
  FROM transactions
  GROUP BY user_id
)

SELECT
  CASE
    WHEN lifetime_value > 1000 THEN 'VIP'
    WHEN lifetime_value > 500 THEN 'Premium'
    ELSE 'Standard'
  END AS segment,
  COUNT(*) AS users,
  AVG(orders) AS avg_orders,
  AVG(avg_rating) AS satisfaction
FROM user_metrics
GROUP BY segment
ORDER BY lifetime_value DESC;

Breaking Into the Industry

The most successful career transitions combine:

  1. 3-5 substantial portfolio projects with clear business impact
  2. Contributions to open-source data projects (scikit-learn, pandas)
  3. Networking through meetups and Kaggle competitions
  4. Specialization in a high-demand niche (NLP, computer vision, etc.)

Pro Tip: Start with data analyst roles to gain domain knowledge before transitioning to full data science positions.
